# Repository Instructions

Use the local skill at `skills/history-to-md-maintainer/SKILL.md` for any work in this repository that includes:

- commit messages
- change summaries
- review summaries
- handoff notes

When a fact should be parseable later from git history or markdown, use the exact `H2M-<TAG>:` wording from the skill. Do not use decorative emphasis like `!!! ... !!!` for durable project rules.
